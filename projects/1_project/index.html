Handwritten recognition

This pet project, I will demonstrate for you guys a really exciting example of using Neural Network to predict handwritten digits. You could draw on the canvas displayed on your screen and inference with the pre-train model. We use \(Tensorflowjs\) to convert Python pre-train model to tfjs format. Let's first start with the theoretical of the net.

What are neurons?

Neurons network are inspired by the brain, what are neurons and in what sense are they linked together? Neuron is a thing that holds a number. Specifically, a number between 0 and 1, and it's really not more than that.

For example, the network starts with a bunch of neurons corresponding to each other of \(28\times 28 = 784\) pixels of the input image. Which is \(784\) neurons in total, each of those holds a number that represents the gray scale value of corresponding pixel. ranging from 0 for while pixels and 1 for black pixels.

This number inside the neuron is called its \(activation\). All the \(784\) neurons make up the first layer of neural network. The last layer has ten neurons each representing one of the digits. The activation in these neurons again some number that between 0 and 1. Represents how much the system thinks that a given image correspond with a given digit. There's also a couple of layers in between called the hidden layers, inside these layer, we will answer the question how on earth is process of recognizing digits is going to be handle.

Citations

Citations are then used in the article body with the <d-cite> tag. The key attribute is a reference to the id provided in the bibliography. The key attribute can take multiple ids, separated by commas.

The citation is presented inline like this: (a number that displays more information on hover). If you have an appendix, a bibliography is automatically created and populated in it.

Distill chose a numerical inline citation style to improve readability of citation dense articles and because many of the benefits of longer citations are obviated by displaying more information on hover. However, we consider it good style to mention author last names if you discuss something at length and it fits into the flow well — the authors are human and it’s nice for them to have the community associate them with their work.

Footnotes

Just wrap the text you would like to show up in a footnote in a <d-footnote> tag. The number of the footnote will be automatically generated.This will become a hoverable footnote.
